## DMIT 换 IP 的几种方式，先搞清楚你属于哪种情况

换 IP 之前，先确认一件事：你的 IP 是"被墙了"还是"想换一个更好的"？这两种情况对应的操作路径不太一样。

### 情况一：IP 被 GFW 封锁（最常见）

这是大多数人换 IP 的原因。判断方法很简单——在国内用 ping 或者 tcping 测一下你的 IP，如果完全不通，但在境外节点上能正常访问，基本就是被墙了。

DMIT 对于 IP 被墙的处理方式，不同套餐有差异：

- **PVM 系列（Premium 线路，比如 CN2 GIA、CMIN2）**：IP 被墙后可以申请更换，但通常是收费的，价格视套餐而定。部分套餐有一次免费更换的额度，用完之后就要付费。
- **Eyeball 系列**：面向普通用户的套餐，IP 被墙的处理政策相对宽松一些，具体以工单回复为准。
- **Pro 系列**：高端线路，IP 资源更稀缺，换 IP 的费用相对更高。

### 情况二：主动申请更换 IP（IP 没被墙，但想换）

这种情况 DMIT 一般是直接收费处理，不存在免费额度。费用通过工单确认，付款后后台操作。

### 情况三：重装系统顺带换 IP

重装系统本身不会自动换 IP。IP 是绑定在实例上的，不是绑定在系统上的。这个误区很多新手会踩。

## 后台操作步骤：从提交工单到 IP 生效

DMIT 换 IP 没有一键按钮，必须通过工单（Support Ticket）来操作。下面是完整流程：

### 第一步：登录后台，确认实例信息

登录 DMIT 后台，进入「Services」→「My Services」，找到你要换 IP 的实例，记下以下信息：
- 服务 ID（Service ID）
- 当前 IP 地址
- 套餐名称

这些信息在提交工单时要用到，写清楚能加快处理速度。

### 第二步：提交工单

进入「Support」→「Open New Ticket」，选择对应的部门（一般选「VPS Support」）。

工单内容建议这样写（直接套用）：

> Hello,  
> I would like to request an IP change for my VPS.  
> Service ID: [你的 ID]  
> Current IP: [当前 IP]  
> Reason: IP is blocked in mainland China / I would like a clean IP  
> Please advise on the availability and any associated fees.  
> Thank you.

写英文回复更快，DMIT 客服团队主要用英文处理工单。中文也能看懂，但响应速度可能慢半拍。

### 第三步：等待确认并付款

客服会回复告知是否有可用 IP、费用是多少。确认后按指引付款（通常是在后台生成一张 Invoice）。

### 第四步：IP 更换生效

付款完成后，客服会在后台操作更换 IP。整个过程实例会有短暂的网络中断，一般几分钟内恢复。新 IP 生效后记得更新你本地的配置文件（SSH config、代理配置等）。

## 换 IP 之前，这几件事先做

很多人换完 IP 发现新 IP 用了没多久又被墙，根本原因是没有解决"为什么 IP 会被墙"这个问题。换 IP 只是治标，下面这些才是治本的操作：

### 检查你的流量特征

如果你在用 Shadowsocks、V2Ray 这类工具，协议特征太明显是 IP 被墙的主要原因之一。换 IP 之前先考虑换协议或者加混淆，不然新 IP 照样撑不了多久。

### 避免 IP 被大量扫描

有些人拿 VPS 跑端口扫描、爬虫、发邮件，这些行为会加速 IP 进黑名单。换了新 IP 之后，这些操作要么停掉，要么换个更合适的服务商来跑。

### 确认新 IP 的质量

换到新 IP 之后，建议用 IP 质量检测工具（比如 ipinfo.io、scamalytics）查一下新 IP 的风险评分，以及是否在常见黑名单里。DMIT 的 IP 资源整体质量不错，但也不是每个 IP 都是干净的。

## FAQ：换 IP 常见问题

### DMIT 换 IP 要多少钱？

没有统一定价，需要通过工单询价。根据套餐不同，通常在 $5 到 $15 之间，高端套餐（Pro 系列）可能更高。部分套餐在特定条件下有一次免费更换额度，具体以工单回复为准。

### 换 IP 之后原来的数据会丢失吗？

不会。换 IP 只是更换实例的网络出口地址，系统、数据、配置文件全部保留。唯一需要手动更新的是你本地保存的 IP 地址（SSH 配置、代理客户端配置等）。

### 工单多久能处理？

DMIT 的工单响应速度在独立 VPS 服务商里算比较快的，工作时间内通常 2–6 小时内有回复。非工作时间可能要等到次日。用英文写工单会快一些。

### 可以换到指定的 IP 段吗？

一般不行。DMIT 会从可用 IP 池里分配，不支持指定具体 IP 或 IP 段。如果对 IP 质量有特殊要求，可以在工单里说明，客服会尽量匹配，但不保证。

### 重装系统能换 IP 吗？

不能。IP 绑定在实例上，重装系统不影响 IP。想换 IP 只能走工单流程。

### DMIT 支持哪些付款方式？

支持 PayPal、信用卡、支付宝、加密货币等多种方式，覆盖国内外用户的主流需求。

### 换 IP 之后需要重新配置什么？

主要是更新你本地的连接配置：SSH config 里的 Host 地址、代理客户端里的服务器 IP、以及任何绑定了旧 IP 的 DNS 记录或防火墙规则。
